pgb-viewer-test

This is a page for playing around with embedding windows to Google Docs in a Gh page using an i-frame. The page is displaying using the .io page. I'm not sure yet how to incorporate this into our practical display of the PGB file.

Here is what I have learned so far.

  • For the Google doc to properly display on Github, sharing must be set so that anyone with the link can see the file.
  • It must also be set to share to the web using File/Share/Publish to web/Embed. An iframe tag will be created for you to use.
  • I will play around more with the frame controls, but adding width="100%" and height="600" to the iframe gives a window that runs the full width of the screen
  • Also set the embed doc to automatically republish if the data changes.
  • You can set up the Google sheet to just show one page or the whole workbook.
  • It takes a few minutes for the changes to ripple through.

All in all, I think we can make this work.
